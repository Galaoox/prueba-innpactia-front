## Instalacion

Pasos para la instalacion de dependencias:

```bash
  cd prueba-innpactia-front
  yarn install
```

# Ejecucion



Pasos para ejecutar el proyecto:

```bash
  cd prueba-innpactia-front
  ng serve
```
